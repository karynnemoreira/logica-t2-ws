Objetos {}

Os objetos são estruturas de dados que nos permitem armazenar dados e funcionalidades. Não são ordenadas.

Array []

Array são estruturas semelhantes a listas e podem conter diversos tipos de dados, inclusive podemos guardar objetos {} dendo de um array. São ordenadas. Array trabalha com índice numérico [0,1,2,3,4...]

Métodos de um array

Método é um gatilho de um função que já tem uma funcionalidade embutida. Métodos de inclusão e exclusão de itens.
