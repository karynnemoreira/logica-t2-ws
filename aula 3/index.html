Objetos

Os objetos são estruturas de dados que nos permitem armazenar dados e funcionalidades utilizando chaves { }

  • Os objetos não garantem uma ordem específica. Não são ordenados.
  • Pode conter múltiplos tipos de dados.
  • Utilizamos chaves {} para sua declaração.
  • As propriedades/elementos, são acessados usando ponto .

Array

Array são estruturas semelhantes a listas e podem conter diversos tipos de dados. Inclusive guardar objetos dentro de um array.

  • Os elementos em um array são ordenados e cada um tem um índice numérico. Começando do zero 0 [ 0 1 2 3 ...]
  • Pode conter múltiplos tipos de dados
  • Utiliza colchetes [ ]
  • Os elementos são acessados usando o índice numérico

Métodos de array

  • Push, ele adiona um item no final da array
  • Unshift, adiciona um item no começo da array
  • Pop, remove o último item da array
  • Shift, remove o primeiro item da array
