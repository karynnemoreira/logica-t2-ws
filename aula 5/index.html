Function

Uma função é um bloco de código que realiza uma tarefa específica e pode ser reutilizado.

Funções podem receber dados externos através de parâmetros() Esses dados são como variáveis que a função utliza ao ser chamada.

Parâmetros e Argumentos

Os parâmetros de uma função são os valores que a função espera receber.

Os argumentos são os valores que são passados para uma função, preenchendo os marcadores de posição definidos pelo parâmetros.

Função anônima

é uma função que não tem nome, ou seja, você cria uma função, mas não dá um nome pra ela.

Arrow Function

É uma expressão de função de seta. É uma alternativa compacta a expressão de função tradicional.
