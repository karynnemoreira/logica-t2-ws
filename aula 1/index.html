Variáveis

Let: Declara uma variável (Escopo local, é declarada dentro de uma função) Ela pode ser redeclarada, pode mudar o valor.

Const: Não pode ser atualizada, tem o mesmo valor (é constante) Use const para valores que não mudam.

Var: Podem ser atualizadas. É uma variável antiga, menos utilizada atualmente. (Escopo global, é declarada fora de uma função)

Escopo Global, variável fora de uma função let nome = "Samuca" console.log(nome)

Escopo local, variável dentro de uma função function amigo (){ let variavel = "Silverio" console.log(variavel) } amigo()
